```

│   │   └── dedup.py         # 去重中间件
│   ├── pipelines/            # 数据管道
│   │   ├── cleaning.py      # 数据清洗
│   │   ├── storage.py       # 数据存储（MongoDB+PostgreSQL）
│   │   └── images.py        # 图片下载（MinIO）
│   ├── items.py             # Item 定义
│   ├── settings.py          # 配置（6.7KB）
│   └── utils/               # 工具函数
│       ├── logging_config.py # 日志配置
│       ├── metrics.py       # Prometheus 监控
│       └── simhash.py       # SimHash 去重
├── config/
│   └── sites.yaml           # 站点配置（4.5KB）
├── tests/
│   └── test_base.py         # 单元测试
├── logs/                    # 日志目录
├── scrapy.cfg               # Scrapy 配置
└── requirements.txt         # Python 依赖
```

#### 2. 核心功能实现

**Scrapy 配置（settings.py）：**
- 并发设置：16 全局，2 每域名
- 下载延迟：1 秒
- 自动节流：启用
- 重试配置：3 次重试
- 中间件：UA 轮换、代理、重试、去重
- Pipeline：清洗、去重、存储、图片

**中间件：**
- UserAgentRandomizerMiddleware：15+ UA 轮换
- ProxyMiddleware：代理池管理、质量监控
- CustomRetryMiddleware：指数退避重试
- DedupMiddleware：Redis Bloom Filter 去重

**Pipeline：**
- CleaningPipeline：Readability 正文提取、质量评分
- MongoDBPipeline：原始 HTML 存储
- PostgreSQLPipeline：结构化数据存储
- ImageDownloadPipeline：图片下载、MinIO 上传、缩略图生成

**工具函数：**
- logging_config：日志轮转配置
- metrics：Prometheus 监控指标
- simhash：SimHash 内容指纹
