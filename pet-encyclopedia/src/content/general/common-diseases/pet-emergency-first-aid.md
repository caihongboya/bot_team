# 宠物常见急症处理

## 中毒

### 症状
- 呕吐、腹泻
- 抽搐、颤抖
- 呼吸困难
- 意识模糊

### 处理
1. 立即移除毒物
2. 不要催吐（除非兽医指导）
3. 带上毒物包装就医
4. 联系兽医或毒物控制中心

### 常见毒物
- 巧克力
- 葡萄/葡萄干
- 洋葱、大蒜
- 木糖醇
- 杀虫剂
- 人类药物

## 外伤出血

### 处理
1. 用干净纱布按压伤口
2. 持续按压 5-10 分钟
3. 如渗透，加层不移除
4. 止血后包扎
5. 严重出血立即就医

## 骨折

### 症状
- 跛行
- 疼痛
- 肿胀
- 异常姿势

### 处理
1. 限制活动
2. 不要试图复位
3. 用夹板固定（如会）
4. 立即就医

## 窒息

### 症状
- 呼吸困难
- 抓挠嘴巴
- 牙龈发紫

### 处理
1. 打开口腔检查
2. 如可见异物，小心取出
3. 海姆立克急救法（小型宠物）
4. 立即就医

## 中暑

### 症状
- 过度喘气
- 流口水
- 虚弱
- 体温升高

### 处理
1. 移到阴凉处
2. 用温水（不是冷水）降温
3. 提供饮水
4. 如严重，立即就医

## 癫痫发作

### 处理
1. 保持冷静
2. 移开周围危险物品
3. 不要把手放入口中
4. 记录发作时间
5. 如超过 5 分钟，立即就医

## 烧伤

### 处理
1. 用凉水冲洗 10-15 分钟
2. 不要涂抹药膏
3. 用干净纱布覆盖
4. 就医

## 急救箱必备

- 纱布、绷带
- 消毒液
- 剪刀
- 镊子
- 体温计
- 止血粉
- 宠物专用药物
- 兽医联系方式
