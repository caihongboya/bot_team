## 三、开发 Spider

### 3.1 基础 Spider 类

所有 Spider 应继承 `BaseSpider`：

```python
from crawler_core.spiders.base import BaseSpider
from crawler_core.items import PetArticleItem

class MySpider(BaseSpider):
    name = 'my_spider'
    allowed_domains = ['example.com']
    start_urls = ['https://example.com']
    
    custom_settings = {
        'DOWNLOAD_DELAY': 1,
        'CONCURRENT_REQUESTS_PER_DOMAIN': 2,
    }
    
    def parse(self, response):
        # 提取数据
        item = PetArticleItem()
        item['title'] = response.css('h1::text').get()
        item['content'] = response.css('.content::text').get()
        yield item
```

### 3.2 选择器使用

```python
# CSS 选择器
title = response.css('h1.title::text').get()
links = response.css('a[href*="/article/"]::attr(href)').getall()

# XPath 选择器
title = response.xpath('//h1[@class="title"]/text()').get()

# 正则提取
import re
price = re.search(r'¥(\d+)', text).group(1)
```

### 3.3 请求和回调

```python
# 跟随链接
yield response.follow(
    url=link,
    callback=self.parse_detail,
    meta={'category': 'cat'},
)

# 发送新请求
yield scrapy.Request(
    url='https://api.example.com/data',
    callback=self.parse_api,
    headers={'Authorization': 'Bearer token'},
)
```

## 十、常见问题

### Q1: 如何调试 Spider？

```bash
# 使用 Scrapy Shell
scrapy shell 'https://example.com'

# 在 shell 中测试选择器
response.css('h1::text').get()
```

### Q2: 如何处理 JavaScript 渲染？

使用 Playwright 中间件：

```python
# 在 Spider 中启用
custom_settings = {
    'DOWNLOAD_HANDLERS': {
        'https': 'scrapy_playwright.handler.ScrapyPlaywrightDownloadHandler',
    },
}

# 在请求中使用
yield scrapy.Request(
    url,
    meta={'playwright': True},
)
```

### Q3: 如何恢复中断的爬取？

使用 HTTP 缓存：

```bash
# 启用缓存后，断点续传会自动使用缓存
scrapy crawl my_spider -s HTTPCACHE_ENABLED=1
```

### Q4: 如何导出爬取结果？

```bash
# JSON 格式
scrapy crawl my_spider -o output.json

# CSV 格式
scrapy crawl my_spider -o output.csv

# JSONL 格式
scrapy crawl my_spider -o output.jl
```
