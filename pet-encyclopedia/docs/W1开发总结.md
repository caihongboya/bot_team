## 2. 数据库 Schema

### 2.1 表结构

| 表名 | 说明 | 字段数 |
|------|------|--------|
| categories | 分类表 | 8 |
| articles | 文章表 | 12 |
| tags | 标签表 | 3 |
| article_tags | 文章标签关联表 | 2 |
| article_search | 文章搜索索引表 | 2 |

### 2.2 核心功能

- **分类层级**: 支持两级分类（一级 + 二级）
- **全文搜索**: PostgreSQL tsvector + GIN 索引
- **自动更新**: 触发器自动维护搜索索引和更新时间
- **初始数据**: 13 个分类（3 个一级 + 10 个二级）

### 2.3 文件位置

`src/lib/schema.sql`

## 3. API 设计

### 3.1 API 端点

| 方法 | 路径 | 说明 | 状态 |
|------|------|------|------|
| GET | /api/v1/articles | 获取文章列表 | ✅ 已实现 |
| GET | /api/v1/articles/:slug | 获取文章详情 | ✅ 已实现 |
| GET | /api/v1/categories | 获取分类列表 | ✅ 已实现 |
| GET | /api/v1/search?q= | 搜索文章 | ✅ 已实现 |

### 3.2 功能特性

- **分页**: 支持 page 和 limit 参数
- **筛选**: 支持按分类、状态筛选
- **搜索**: PostgreSQL 全文搜索 + 关键词高亮
- **相关度**: 相关文章推荐

`docs/API 设计文档.md`

## 4. 开发环境配置

### 4.1 系统要求

- Node.js 20.x
- PostgreSQL 15+
- npm 10.x

### 4.2 快速开始

```bash
# 安装依赖
npm install

# 创建环境变量
cp .env.local.example .env.local
# 编辑 .env.local 配置 DATABASE_URL

# 初始化数据库
npm run db:init

# 导入示例内容
npm run content:import

# 启动开发服务器
npm run dev
```
