## 一、爬虫系统架构设计

### 1.1 分布式爬虫框架选型

#### 推荐方案：Scrapy + Playwright 混合架构

| 框架 | 适用场景 | 优势 | 劣势 |
|------|----------|------|------|
| **Scrapy** | 静态 HTML、API 接口、大批量爬取 | 高性能、异步、扩展性强、中间件丰富 | 对 JavaScript 渲染支持弱 |
| **Playwright** | 动态渲染、SPA 应用、复杂交互 | 支持现代浏览器、可处理 JS 渲染、截图能力 | 资源消耗大、速度较慢 |
| **Puppeteer** | Chrome 专用场景 | 轻量、Chrome 官方支持 | 仅支持 Chrome、生态不如 Playwright |

#### 架构设计

```
┌─────────────────────────────────────────────────────────────┐
│                      调度中心 (Scheduler)                    │
│  - 任务队列管理  - URL 去重  - 优先级调度  - 分布式协调       │
└─────────────────────────────────────────────────────────────┘
                              │
              ┌───────────────┼───────────────┐
              │               │               │
              ▼               ▼               ▼
    ┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐
    │  Scrapy Worker  │ │  Scrapy Worker  │ │ Playwright Worker│
    │   (静态站点)    │ │   (静态站点)    │ │   (动态站点)    │
    └─────────────────┘ └─────────────────┘ └─────────────────┘
              │               │               │
              └───────────────┼───────────────┘
                              ▼
    ┌─────────────────────────────────────────────────────────┐
    │                    数据存储层                             │
    │  - Redis (缓存/队列/去重)  - MongoDB (原始 HTML)          │
    │  - PostgreSQL (结构化内容)  - MinIO/S3 (图片存储)        │
    └─────────────────────────────────────────────────────────┘
```

#### 技术栈选型

| 组件 | 选型 | 理由 |
|------|------|------|
| 爬虫框架 | Scrapy 2.11+ | 成熟稳定、生态丰富、高性能异步 |
| 动态渲染 | Playwright (Python) | 多浏览器支持、API 友好、微软维护 |
| 任务队列 | Redis + RQ/Celery | 简单可靠、支持优先级、易监控 |
| URL 去重 | Redis Bloom Filter | 空间效率高、支持分布式 |
| 原始存储 | MongoDB | 文档型、灵活 schema、适合 HTML 存储 |
| 结构化存储 | PostgreSQL | 关系型、ACID、便于查询分析 |
| 图片存储 | MinIO (自建 S3) | 兼容 S3 协议、自建可控、成本低 |

```python
    # 1. 转为小写
    url = url.lower()
    # 2. 移除 fragment
    url = url.split('#')[0]
    # 3. 标准化查询参数（排序）
    # 4. 移除跟踪参数 (utm_*, fbclid, etc.)
    # 5. 统一协议 (http/https)
    # 6. 统一 www 前缀

# Bloom Filter 配置
BLOOM_FILTER_CONFIG = {
```

```python
def extract_images(html, base_url):
    soup = BeautifulSoup(html, 'lxml')
    images = []
```

```python
from PIL import Image
import io

def process_image(image_bytes, max_width=1920, quality=85):
    img = Image.open(io.BytesIO(image_bytes))
    
    # 格式转换（转 WebP）
    if img.mode in ('RGBA', 'LA', 'P'):
        output_format = 'PNG'
    else:
        output_format = 'WebP'
    
    # 尺寸调整
    if img.width > max_width:
        ratio = max_width / img.width
        new_height = int(img.height * ratio)
        img = img.resize((max_width, new_height), Image.LANCZOS)
    
    # 压缩输出
    output = io.BytesIO()
    img.save(output, format=output_format, quality=quality, optimize=True)
    
    return output.getvalue(), output_format.lower()
```

### 2.4 敏感词过滤

#### 过滤方案

**方案：DFA 算法 + 自定义词库**

```python
class SensitiveWordFilter:
    def __init__(self, word_list):
        self.keyword_chains = {}
        self.build_dfa(word_list)
    
    def build_dfa(self, word_list):
        """构建 DFA 状态机"""
        for word in word_list:
            level = self.keyword_chains
            for char in word:
                if char not in level:
                    level[char] = {}
                level = level[char]
            level['end'] = True
    
    def filter(self, text, replace_char='*'):
        """过滤敏感词"""
        result = []
        i = 0
        while i < len(text):
            level = self.keyword_chains
            found = False
            for j in range(i, len(text)):
                if text[j] not in level:
                    break
                level = level[text[j]]
                if 'end' in level:
                    # 发现敏感词
                    result.extend([replace_char] * (j - i + 1))
                    i = j + 1
                    found = True
                    break
            if not found:
                result.append(text[i])
                i += 1
        return ''.join(result)
```

```python
from rq import Queue, Worker

# 初始化
redis_conn = Redis(host='localhost', port=6379, db=0)
pending_queue = Queue('pet:queue:pending', connection=redis_conn)
result_queue = Queue('pet:queue:result', connection=redis_conn)

# Worker 配置
worker = Worker(
    [pending_queue],
    connection=redis_conn,
    log_job_description=True,
    job_monitoring_interval=10,
)

# 启动 Worker
worker.work(burst=False)
```
