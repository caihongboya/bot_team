### 任务 3：性能优化

#### 3.1 图片懒加载

**实现方式：**
```html
<img 
  src={article.coverImage} 
  alt={article.title}
  loading="lazy"
  decoding="async"
/>
```

**应用范围：**
- 文章详情页封面图
- 文章列表封面图（待添加）
- 分类页封面图（待添加）

**效果：**
- 首屏加载速度提升
- 减少初始带宽消耗
- 改善用户体验

#### 3.2 缓存策略

**分类 API（`/api/v1/categories`）：**
```typescript
export const revalidate = 300; // 5 分钟 ISR
```
- 分类数据不常变化
- 5 分钟缓存平衡实时性和性能

**文章列表 API（`/api/v1/articles`）：**
```typescript
export const revalidate = 120; // 2 分钟 ISR
```
- 文章列表较频繁更新
- 2 分钟缓存

**搜索 API（`/api/v1/search`）：**
- 不缓存，实时查询
- 保证搜索结果最新

**文章详情页：**
```typescript
{ cache: 'no-store' }
```
- 动态渲染，保证实时性
- 浏览量统计需要实时更新

#### 3.3 数据库索引

**已有索引：**
- `idx_categories_parent`：父分类查询
- `idx_categories_slug`：分类 slug 查询
- `idx_articles_category`：分类文章查询
- `idx_articles_status`：状态筛选
- `idx_articles_slug`：文章 slug 查询
- `idx_articles_created`：发布时间排序
- `idx_article_tags_article`：文章标签查询
- `idx_article_tags_tag`：标签文章查询
- `idx_article_search_vector`：搜索向量 GIN 索引
